第32题

把数组排成最小的数：输入一个正整数数组，把数组里所有数字拼接起来排成一个数，打印能拼接出的所有数字中最小的一个。例如输入数组{3，32，321}，则打印出这三个数字能排成的最小数字为321323。

python

思路：

java

思路：先把数组的元素复制到集合里面，然后用Collections的sort方法进行排序，其中要传入比较器要加泛型所以不能直接用数组，必须用集合。

String的compareTo方法比较两个string类型数字时候，是从左到右一位一位的比较acsii码值然后返回，如果前面几位都相同就返回长度的差值
例如：”222”.compareTo(“555”)返回-3
“222”.compareTo(“111111”)返回1
“222”.compareTo(“222333”)返回-3

在本题中，要比较的两个数字位数是相同的，所以str1小于str2就返回负数，str1大于str2就返回整数，str1与str2就返回0。
比较两个字符串str1,str2大小的时候，先将它们拼接起来，比较s1和s2那个大，如果s1大，那说明str2应该放前面，所以按这个规则，str2就应该排在str1前面。
例如：str1=3 str2=32 拼接后 s1=332 s2=323 s1>s2 所以str2应该放前面，得到的拼接字符串s2(323)才是最小的。（题目要求的是组成最小数字）

第36题

两个链表的第一个公共结点：输入两个链表，找出它们的第一个公共结点。

python

思路：

java

思路：如遇到一个结点相同，则这个节点后面的结点都相同。分别计算出两个链表的长度，然后遍历较长的那个链表，遍历两个链表的长度之差次。然后一起遍历，直到遇到相同的结点就是第一个公共结点，就可以返回了
