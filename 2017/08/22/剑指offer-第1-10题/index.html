剑指offer-第1-10题

发表于 2017-08-22 |

剑指offer第1-10题，分别用java和python解答并且有解题思路。

第1题

二维数组的查找：在一个二维数组中，每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

python

思路：利用(每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序)这个条件
从左下角或者右上方开始遍历。因为如果从左上角开始遍历，那么右边和下边的值都比他大，不能确定往哪走。右下角同
理。而从左下角开始，那么整数比这个元素大的话就往右找，比这个元素小的话往上找。这样就可以使比较的次数最少。

第2题

替换空格：请实现一个函数，将一个字符串中的空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。

python

思路：

  1. 转换成字符串
  2. 再转成char数组(也可以不转)
  3. 再构造stringbuffer对象(string对象也行)
  4. 再转换成字符串。

第3题

从头到尾打印链表：输入一个链表，从尾到头打印链表每个节点的值。

python

思路：先用python写了下，你麻痹虽然思路对了，毕竟这题还是基础题。先判断节点是否为空，不空就添加节点的值到列表第一位，然后节点指向下一个；为空就直接返回空列表。重复上面操作。但好多地方写错，小毛病太多。比如：写了i++我艹，最近java写多了，python语法都忘了，None写成了none和null。

java

思路：更是日了狗了，多写个右括号尼玛找了好久，跟python思路一样，目测是最简单的方法了，

别人代码感觉都好麻烦。都是什么递归，堆栈之类的。

第4题

重建二叉树：输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

python

思路：跟java一样，递归时候传的列表不用那么麻烦的创建了。

java

思路：第一次做，自己写的代码，占用了2万多k的内存，别人好多都是0…tony马
虽然也是用的递归，我是先把索引找到然后把再把左右子树的前序和中序遍历重新传给本函数。

第5题

用两个栈实现队列：用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。

python

思路：跟java一样的思路，就是老忘self错了不少次

java

思路：入队列就直接入，出队列：栈1入队列，栈2缓存先判断两个栈是否都为空，为空就抛异常，若不全为空，那么判断栈2是否为空，为空就把栈1元素推进栈2再从栈2出栈，不为空就直接从栈2出栈。搞不懂为什么!stack1.empty()这句换成stack1.peek()!=null这句不能用，栈为空的时候就报错，peek()方法不是查看栈顶对象吗，为空就返回null，不知道哪里有问题。

python

思路：第一个值是数组递增最小的，从前到后找到比这个小的数就返回不用往下找了

java

思路：跟python相同，结果时空复杂度都比python少一大截
