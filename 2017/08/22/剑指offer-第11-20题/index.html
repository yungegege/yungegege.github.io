剑指offer第11-20题，分别用java和python解答并且有解题思路。

第11题

二进制中1的个数：输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

python

思路：位运算

  1. “>> 右移,高位补符号位” 这里右移一位表示除2
  2. “>>> 无符号右移,高位补0”；
  3. “<< 左移” 左移一位表示乘2，二位就表示4，就是2的n次方

java

思路：如果一个整数不为0，那么这个整数至少有一位是1。如果我们把这个整数减1，那么原来处在整数最右边的1就会变为0，原来在1后面的所有的0都会变成1(如果最右边的1后面还有0的话)。其余所有位将不会受到影响。
举个例子：一个二进制数1100，从右边数起第三位是处于最右边的一个1。减去1后，第三位变成0，它后面的两位0变成了1，而前面的1保持不变，因此得到的结果是1011.我们发现减1的结果是把最右边的一个1开始的所有位都取反了。这个时候如果我们再把原来的整数和减去1之后的结果做与运算，从原来整数最右边一个1那一位开始所有位都会变成0。如1100&1011=1000.也就是说，把一个整数减去1，再和原整数做与运算，会把该整数最右边一个1变成0.那么一个整数的二进制有多少个1，就可以进行多少次这样的操作。

第15题

反转链表：输入一个链表，反转链表后，输出链表的所有元素。

python

思路：遍历每个结点并保存到list，再反转后取出，链表为空就直接返回空

java

思路：每遍历一个结点就执行以下操作：
1.把next指向下一个结点 （把下个节点保存下来，放着断裂）
2.这个结点的下一个结点改为pre
3.pre改为这个结点
4.这个结点改为next指向的下一个结点
到最后的pre就指向了链表最后一个结点也就是反转后的第一个结点再返回就行了。有点绕但程序比上面的简单得多，不用循环两次，循环结束就是程序结束。

第16题

合并两个排序的链表：输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

python

思路：先写1个为空或者2个为空的情况。创建一个开始节点，用p保存。然后都不为空时循环以下操作:如果结点1大于结点2的值,开始结点的next指向小的那个也就是2，2指向2的下一个，开始结点指向开始结点的next。直到有一个为空时结束循环，1或者2哪个为空开始结点的next结点就指向那个。最后，因为p在合成链表的前面，所以要返回p的下一个节点也就是合成链表的第一个结点。

java

思路：不能new新的结点只能等于null。所以这几点要注意下，在while循环要判断下p是否为空，第一次循环时为空就把第一个结点值赋给它。不为空就把node的下一个执行list，node再指向node的下一个，然后不管为不为空循环一次list都执行它的他一个了。最后，返回p跟上面有点区别，因为这里的p就是合成链表第一个节点。
